Welcome to my personal portfolio website! This is a premium, ultra-modern, fully responsive developer portfolio built with React, Vite, Tailwind CSS, and Framer Motion. 

It highlights my experience as an AI & Data Science B.Tech student at MITS-DU and a Full Stack MERN developer.

## ✨ Features

- **🛡️ Floating Glassmorphic Navbar**: A sticky header featuring custom branding ("Varun | Full Stack Developer") and smooth hover underlining animations with neon gradients.
- **📱 Fully Responsive Design**: Optimized to render seamlessly across all screen sizes (mobile smartphones, tablets, laptops, and large desktop screens) with stacked layouts and mobile toggle navigation.
- **✨ Interactive Motion & Tilt**: Uses `framer-motion` for smooth scroll fade-ins and interactive magnet physics on the portrait.
- **📁 Deployed Project Integrations**: Features real screenshots and live deploy links for active projects:
  - **Bazaar India**: AI-powered E-Commerce recommendation system on Render.
  - **Royal Glow Studio**: Luxury beauty salon booking marketplace on Vercel.
  - **Image Gallery**: Minimalist image grid application on GitHub.
  - **Responsive Portfolio & Practice projects**.
- **⚙️ Serverless SSR Backend**: Powered by Nitro server engine running seamlessly on Vercel Serverless Functions.

## 💻 Getting Started Locally

Follow these steps to run the project locally on your machine:

### Prerequisites
Make sure you have Node.js installed.

### Installation
1. Clone this repository to your local directory.
2. Open your terminal in the project folder and run:
   ```bash
   npm install
   ```

### Running Locally
To launch the development server:
```bash
npm run dev
```
Open **[http://localhost:8080](http://localhost:8080)** in your browser to view the application.

### Building for Production
To generate Vercel-compatible serverless build outputs:
```bash
$env:NITRO_PRESET="vercel"
npm run build
```
